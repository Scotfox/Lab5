Завдання 6: Форматування програмного коду

Для правильного відображення коду використовуються спеціальні семантичні теги.

Наприклад, у формулі розрахунку площі кола (S = π * r2), тег <var> використовується для позначення змінної r (радіус).

Нижче наведено приклад коду (теги <pre> та <code>): 


function calculateArea(r) 
{
    let S = Math.PI * Math.pow(r, 2);
    return S;
}

Завдання 12: Дослідження нових семантичних тегів

1. Тег <mark>

Цей тег використовується для візуального виділення тексту, на який потрібно звернути увагу користувача (як маркер у книзі).

2. Тег <progress>

Відображає прогрес виконання якоїсь задачі. Наприклад, стан завантаження файлу:

Завантажено: 75%

3. Тег <meter>

Використовується для відображення значення в межах відомого діапазону. Наприклад, рівень заряду батареї або місце на диску:

Заповненість диска: 80%

4. Тег <output>

Позначає результат обчислення. Зазвичай працює в парі з JavaScript або формами. Приклад: 25 + 25 = 50

5. Тег <wbr>

Вказує браузеру місце, де можна безпечно розірвати довге слово, якщо воно не поміщається у рядок на мобільних екранах.
