Дослідження нових тегів(Завдання 11)

Теги <blockquote> та <cite>

Тег <blockquote> використовується для виділення довгих цитат. Тег <cite> вказує на джерело або автора цитати.

"Програмування — це мистецтво перетворювати ідеї на реальність."
—тіпа загадачна

Теги <pre> та <code>

Тег <code> потрібен для відображення коротких фрагментів комп'ютерного коду. Тег <pre> зберігає всі пробіли та перенесення рядків (форматування), що ідеально підходить для великих блоків коду. 


function sayHello() 
{
    console.log("Привіт ворлд!");
}

Тег <address>

Тег <address> містить контактну інформацію автора або власника документа.
